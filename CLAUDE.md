# CLAUDE.md

## Project

Game catalog for 123fzero — a curated directory of Flipper Zero games and apps. Contains only markdown files, links, and images. No source code lives here.

## Conventions

- All content in English
- One section per game in README.md
- Each game links to its own repository
- Images go in `images/` directory

## Language

User prefers communication in Russian. All repo content must be in English.

## GitHub Topics SEO

- Prefer existing, popular GitHub topics that match real user search intent, especially `flipperzero`, `flipper-zero`, `flipper-app`, and `flipper-plugins`
- Use topic phrases that mirror how users search in Google and GitHub: "flipper zero apps", "flipper zero plugins", "flipper zero tools", "flipper zero games"
- Favor canonical topic spellings already used by strong repos over inventing niche custom tags
- Keep topics tightly relevant to Flipper Zero apps, plugins, catalogs, and curated resources
- Avoid broad or noisy topics that dilute relevance, such as generic tags like `flipper` or `catalog`
- Prefer a focused set of high-signal topics over random keyword stuffing, even when using many topics
- When updating topics, verify them in GitHub About after changes and keep the set aligned with the repo's current positioning as a curated Flipper Zero catalog
