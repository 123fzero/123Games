# 123 Games

A catalog of Flipper Zero games and apps by [123fzero](https://github.com/123fzero).

## Games

### 123 Dice D&D

D&D dice roller for Flipper Zero with 3D animated sprites. Supports d4, d6, d8, d10, d12, and d20.

- **Firmware:** Momentum
- **Repo:** [123DiceDnD](https://github.com/123fzero/123DiceDnD)
